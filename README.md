# itskai

_kai's personal website :)_

## setup

```bash
# install tools
brew install docker rbenv nodenv pyenv poetry watchman overmind

# install libraries
brew install libvips ffmpeg

# set up environment
git clone git@github.com:hulloitskai/itskai
cd itskai
bin/setup
```
