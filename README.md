# itskai.me

A complete rewrite of my personal website!

## Setup

```bash
# Install tools
brew install --cask docker
brew install rbenv nodenv pyenv yarn watchman overmind

# Set up environment
git clone git@github.com:hulloitskai/itskai.me
cd itskai.me
bin/setup
```
